Sobre o Projeto Link Solidário

O Link Solidário é uma iniciativa desenvolvida como projeto integrador do 3º período do curso de Análise e Desenvolvimento de Sistemas da UNITINS (Universidade Estadual do Tocantins). Nosso principal objetivo é criar uma ponte digital entre comunidades do Tocantins, facilitando a troca e doação de produtos entre moradores, ONGs e associações comunitárias.

Nossa Missão

Combater o desperdício, promover a sustentabilidade e fortalecer os laços comunitários, permitindo que itens em desuso encontrem novos propósitos e ajudem quem precisa. Acreditamos que, ao facilitar a conexão entre quem tem e quem busca, podemos gerar um impacto social positivo significativo.

Como Funciona?

  • Cadastro de Usuários: Moradores, ONGs e associações podem se registrar em nossa plataforma.
  • Anúncio de Produtos: Usuários podem cadastrar produtos que desejam doar ou trocar, fornecendo descrições detalhadas e fotos.
  • Busca e Filtragem: Interessados podem pesquisar produtos por categoria, localização e tipo de transação (doação ou troca).
  • Manifestação de Interesse: Após encontrar um item, é possível manifestar interesse, iniciando a comunicação entre as partes.
  • Gestão de Transações: A plataforma ajuda a acompanhar o status das doações e trocas, garantindo que os itens cheguem ao seu destino.

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

  • Flask: Um microframework web para Python, utilizado para o desenvolvimento do back-end.
  • Jinja2: Um motor de templates para Python, integrado ao Flask para renderização das páginas HTML.
  • Tailwind CSS: Um framework CSS "utility-first" para estilização rápida e responsiva do front-end.
  • MySQL (via Flask-SQLAlchemy): Sistema de gerenciamento de banco de dados relacional para persistência dos dados.
